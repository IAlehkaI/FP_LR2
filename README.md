# 23-ПО-2 Блошкин Алексей ФП ЛР2 
## Vending Machine Tagless Final

Данный проект представляет собой эмулятор вендингового аппарата (автомата по продаже напитков и снеков), написанный на языке Scala с использованием функционального подхода Tagless Final.

## Описание программы

Программа реализует меню, в котором пользователь может:
- Вносить монеты.
- Покупать товары. Доступна возможность использовать идентификатор студента (Student ID) для получения скидки.
- Запрашивать возврат средств (отмена покупки).
- Пополнять запасы товаров (функция для администратора).
- Начинать новый день, что приводит к сбросу лимита на использование студенческих скидок.
- Просматривать внутреннее состояние автомата (Debug).
- Просматривать историю событий (логи).

Конфигурация автомата (цены, скидки, допустимые монеты) инициализируется при запуске и может быть расширена в коде. Архитектура приложения разделена на доменную модель, алгебры эффектов (ввод/вывод, логика автомата) и интерпретаторы этих алгебр с использованием монадического подхода. 

### Конфигурация по умолчанию:
- Товары: `Cola` (100 монет), `Juice` (120 монет), `Water` (50 монет), `Chips` (80 монет).
- Допустимые монеты: 1, 2, 5, 10, 50, 100, 200.
- Допустимые ID студентов: `1111`, `7777`. Скидка: 20%.

## Структура проекта

- `monads/` — содержит определение базового тайпкласса `Monad`, низкоуровневый `IO` для взаимодействия с консолью и `VendingEff` (объединенный стек эффектов).
- `domain/` — модели данных: конфигурация (`Config`) и текущее изменяемое состояние автомата (`VendingMachineState`).
- `algebras/` — интерфейсы бизнес-логики (`VendingMachine`) и ввода-вывода (`Console`), описанные в стиле Tagless Final.
- `interpreters/` — конкретные реализации интерфейсов (алгебр) `LiveConsole` и `LiveVendingMachine`.
- `Main.scala` — точка входа в программу, в которой инициализируются интерпретаторы, начальное состояние и запускается цикл меню.

## Как запустить

Для запуска проекта вам понадобится установленный `sbt` (Scala Build Tool).

1. Откройте терминал в корневой папке проекта.
2. Запустите команду `sbt run`.

Программа запустится и выведет в консоль интерактивное меню управления автоматом. Следуйте инструкциям на экране, вводя соответствующие номера команд.
